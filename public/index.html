  • MultiplayerSet.com

  • MultiplayerSet is a game about pattern matching. The idea is to find sets of three cards that meet certain requirements. Each card has a four properties. The properties are amount (of shapes), shape, color, and shading.

    A set is three cards for which each card has, for each property, three different properties or one uniform property. Two cards that are both red and a third that is green are not a set.

  • How to play

  • If you want to play with friends, hit start a private game in the top left of this page. You will receive a link you can use to invite other people to join.

    If you want to get into a game quickly, hit join a public game. You will be placed into a game that will start as soon as three people are present. You can still invite your friends to join a public game in the same way.

    Once in a game, click cards to build sets. Click on any part of the board to clear your current selection. Type messages and hit enter to talk to the other players in the game.

    Hints

    If you need a hint or don't see a set on the board (which can happen), use the HINT button. If the majority of players do this, the game will reveal one card of a set on the board. If there are none, the game will deal three additional cards.
