MultiplayerSet

  • MultiplayerSet exists thanks to the following open source projects:

    • headJS, by Tero Piirainen
    • jQuery, by John Resig
    • jQuery BBQ, by "Cowboy" Ben Alman
    • jquery.transform.js, Louis-Rémi Babé

    • Mozilla Developer Center
    • nodeJS, by Ryan Dahl

    MultiplayerSet is a realtime multiplayer game written in nodeJS on the server and jQuery on the client.
